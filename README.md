# PaceUp

##  Problema
Corredores têm dificuldade de manter um ritmo constante durante o treino.

##  Solução
O PaceUp usa a música como âncora de ritmo.
O usuário define o pace manualmente e precisa alcançar esse ritmo — a música não se adapta ao corredor.

##  Funcionalidades
- Controle manual de BPM
- Monitoramento de pace
- Integração com música
- Interface focada em corrida

##  Diferencial
Diferente de apps tradicionais, o ritmo é fixo e o corredor se adapta — criando disciplina e consistência.

## Tecnologias
React + TypeScript + Capacitor

##  Como rodar
npm install  
npm run dev
